# automated-supply-chain-scanner
## 🚀 Advanced Features (Beast 4.0)
* **Signature-Based Deep Content Scanning:** Escapes the limits of file extensions. It reads the actual code logic to find hidden dependencies across any text-based file.
* **Transitive Dependency Resolver:** Parses lockfiles (`package-lock.json`, `Pipfile.lock`) to uncover deeply nested sub-libraries that traditional scanners miss.
* **Dynamic Import Catcher:** Smart regex signatures detect dynamically injected or obfuscated imports like `importlib.import_module()`.
* **Zero-Touch Execution:** No GitHub tokens required. Uses raw content extraction to bypass API limits and authentication errors.
* **Enterprise Dashboard:** Generates an automated, highly professional HTML report complete with metric cards and official CVE/GHSA bug tracking.
